## RPC vs REST

RPC는 주로 기능을 중심으로 설계되어 있으며, 개발자가 프로시저를 호출하는 방식에 초점을 둡니다.
REST는 자원을 중심으로 설계되며, 자원에 대한 행위를 표현하기 위해 HTTP 메서드를 사용합니다.
